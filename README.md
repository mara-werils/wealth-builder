# WealthBuilder Pro

Build wealth systematically using this advanced financial planning tool.

## Features
- **Career Growth Modeling**: Simulate salary progression across different career tracks (FAANG, Startup, etc.).
- **Investment Projections**: Visualize compound interest and different market scenarios.
- **Goal Tracking**: track progress towards achievements like buying a home or financial independence.
- **Premium UI**: Dark mode, glassmorphism design, and interactive charts.

## Deployment
This project is configured for deployment on [Vercel](https://vercel.com).
